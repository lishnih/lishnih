Скрипты VBA для Excel для пакетной обработки данных.

Сортировка_заключений_по_дате
2010-08-23
  Перебирает все листы в книге, извлекает из определённой ячейки дату и
  сортирует листы по этой дате

Генерация_Заключений_УЗК_ГО
2010-08-27
  Извлекает данные с каждой строки из листа "_список" и записывает их в
  заключение, которое создаётся из листа "_образец"

Номер_заключения_в_название_ярлыка
2010-09-02
  Перебирает все листы в книге, извлекает из определённой ячейки номер
  заключения и записывает его в название листа

Генерация_Заключений_ВИК_РД
2010-11-13
  Извлекает данные с каждой строки из листа "_список" и записывает их в
  заключение, которое создаётся из листа "_образец"
  Более "современная" реализация макроса Генерация_Заключений_УЗК_ГО
