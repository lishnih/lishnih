Скрипт VBA для Excel для рисования ёжиков в заключениях НК.

Rev.2 Рисование ёжиков для РК.
Rev.3 Рисование ёжиков для РК с форматированием (выделяет дефекты).
Rev.4 Рисование ёжиков для ВИК (теперь все опции передаются в Sub Рисовать_Ёжика).
Rev.5 Рисование ёжиков для УЗК (def_mode опция).
